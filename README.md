<h1> Ascii Adventure.c</h1>

<p>A randomized maze generator which you can traverse to find some BOOTY and also escape a mostly functioning enemy. This project is completely made in c and is to be played in a terminal.</p>
<p>The algorithm used to implement the randomized maze generation is a simple recursive DFS algorithm.</p>
<p>I also plan to implement a maze solving algorithm using the A* algorithm</p>
